## How to Take Screenshots

### For Android:

1. Run the app: `flutter run`
2. Use Android Studio's screenshot tool or device screenshot (Power + Volume Down)
3. Save screenshots to this directory with appropriate names

### For iOS:

1. Run the app: `flutter run`
2. Use Xcode's screenshot tool or simulator screenshot (Cmd + S)
3. Save screenshots to this directory with appropriate names

### For Architecture & Database Diagrams:

You can create diagrams using tools like:
- [Draw.io](https://draw.io)
- [Excalidraw](https://excalidraw.com)
- [Lucidchart](https://lucidchart.com)
- [Figma](https://figma.com)

## Recommended Screenshot Sizes

- Mobile screenshots: 1080 x 2400 px (or actual device resolution)
- Diagrams: 1200 x 800 px minimum
- Banner: 1200 x 400 px

## Tips for Quality Screenshots

1. Use a clean device state (no status bar notifications)
2. Ensure good contrast and readability
3. Show actual app functionality
4. Include varied note examples for home screen
5. Capture both light backgrounds (for note cards) and dark theme
